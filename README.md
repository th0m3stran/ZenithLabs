# 🏋️ 💪 MyTrainer.AI

MyTrainer.AI is a local-first, personalised AI fitness coach built using Retrieval-Augmented Generation (RAG) to 
deliver training, nutrition and wellness guidance. 

By learning from user inputs such as goals, activity levels, and habits, it generates structured plans and adaptive 
recommendations to help users stay consistent and improve over time. The system combines grounded knowledge 
retrieval with intelligent reasoning to provide reliable, context-aware insights.

Powered by Atlas, our adaptive AI agent.

## 🚀 Features 
- Semantic search over health datasets 
- Retrieval-Augmented Generation (RAG)
- Local LLM inference via Ollama (no external API dependency)
- Full-stack application with real-time interaction

## 🛠 Tech Stack 
- **Backend:** Python, FastAPI
- **Frontend:** React, Typescript 
- **Vector Search:** FAISS/Chroma (Planned)
- **LLM:** Ollama (LLaMA3)
