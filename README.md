# ZenithLabs

ZenithLabs is a local-first AI health assistant built using Retrieval-Augmented Generation (RAG) to deliver personalised fitness and wellness insights.

## Features 
- Semantic search over health datasets 
- Retrieval-Augmented Generation (RAG)
- Local LLM inference via Ollama (no external API dependency)
- Full-stack application with real-time interaction

## Tech Stack 
- Backend: Python, FastAPI
- Frontend: React, Typescript 
- Vector Search: FAISS/Chroma 
- LLM: Ollama (LLaMA3)
